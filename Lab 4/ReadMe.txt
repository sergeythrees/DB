Реализовать БД и написать запросы к ней. 
Выбор СУБД лежит на студенте. Номер в списке соответсвует предметной области для реализации.
К соответсвующей предметной области добавить 2 сущности. 
Например к 1 варианту можно добавить: сотрудник гостиницы, должность в гостинице

Необходимо написать/создать sql запросы:
1. INSERT
	1. Без указания списка полей
	INSERT INTO table_name VALUES (value1, value2, value3, ...);
	2. С указанием списка полей
	INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);
	3. С чтением значения из другой таблицы
	INSERT INTO table2 (column_name(s)) SELECT column_name(s) FROM table1;
2. DELETE
	1. Всех записей
	2. По условию
		DELETE FROM table_name WHERE condition;
	3. Очистить таблицу
		TRUNCATE
3. UPDATE
	1. Всех записей
	2. По условию обновляя один атрибут
		UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;
	3. По условию обновляя несколько атрибутов
		UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;
	
4. SELECT в рамках одной таблицы
	1. С определенным набором извлекаемых атрибутов (SELECT atr1, atr2 FROM...)
	2. Со всеми атрибутами (SELECT * FROM...)
	3. С условием по атрибуту (SELECT * FROM ... WHERE atr1 = "")
